# Compiler

### 1) Сборка проекта:

    1.1) mkdir build && cd build
    1.2) cmake ..
    1.3) make
    
### 2) Запуск
    ./Compiler program_file
    
    В stdout выведется результат работы программы, в output.txt - дерево разбора.
